## Start Service

### Configure Environment
```
  JWT_SECRET=256-bit
  REFRESH_TOKEN_SECRET=256-bit
  DATABASE=your-database-name
  DATABASE_USER=your-user-database
  DATABASE_PASSWORD=your-password-database
  DATABASE_HOST=your-host-database
  DATABASE_PORT=your-port-database-open
  HTTPSPORT=your-port-server-https
  HTTPPORT=your-port-server-http
  REQUEST_INTERCEPT=0 or 1
  SAVE_LOGS=0 or 1
```
